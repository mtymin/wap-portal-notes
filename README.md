# wap-portal-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容不针对特定域名或具体网站，仅作为通用页面存档与分发用途。

## 项目简介

本仓库主要用于存放各类独立 HTML 页面，包括但不限于提示页、过渡页、信息页等。每个页面均为自包含的 HTML 文件，不依赖外部服务器或数据库。适合用于测试、演示、临时页面托管等场景。

## 目录说明

```
/
├── pages/        # 存放所有 HTML 页面文件
├── assets/       # 静态资源文件（图片、样式、脚本等）
└── README.md     # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，命名尽量反映页面用途。
- `assets/`：存放页面共用的资源文件，避免重复引用外部链接。

## 页面归档说明

- 所有页面均为纯前端实现，无后端逻辑。
- 页面之间无依赖关系，可单独部署或访问。
- 归档时请确保页面兼容主流浏览器，并尽量避免使用已废弃的 API。
- 如需更新某个页面，请直接替换对应文件，并更新本说明中的相关记录（如有）。

## 维护说明

- 欢迎提交 PR 添加新页面或优化现有页面。
- 添加页面时，请在 `pages/` 下创建独立文件夹，并确保资源文件路径正确。
- 请勿在页面中包含任何形式的跟踪代码、恶意脚本或违反法律法规的内容。
- 本仓库仅作为页面存档使用，不提供任何形式的服务承诺或技术支持。
